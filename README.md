### Instruções para configuração

---

#### Acessar a pasta do projeto

```bash
$ cd teste-client
```

#### Instalar as dependências

```bash
$ npm install
```

#### Criar o arquivo .env

```bash
$ cp .env.example .env
```

#### Executar o projeto

```bash
$ npm run serve
```

#### Acessar o Client

- ##### [http://localhost:8080/](http://localhost:8080/)
